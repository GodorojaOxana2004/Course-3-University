# Лабораторная работа 5 — Ответы на вопросы

### Вопрос 1

```csharp
int b = 6;
F(b);
Console.WriteLine(b);

static void F(int a)
{
    a = 5;
}
```

**Вопросы и ответы:**

1. **Сколько объектов выделится в динамической памяти?**
   **Ответ:** Нисколько. `int` — значимый тип, хранится в стеке, динамическая память не используется.

2. **Сколько ячеек будет выделено под переменные?**
   **Ответ:** Две. Одна для `b` в `Main`, одна для `a` при вызове `F`.

3. **Перезапишет ли `a = 5` переменную `b`?**
   **Ответ:** Нет. `a` — отдельная копия `b`. `b` останется 6.

4. **Почему у F указан `void` как возвращаемый тип?**
   **Ответ:** Функция не возвращает значение, она только выполняет действия с локальной копией переменной.

### Вопрос 2

```csharp
int a = 6;
F(a);
Console.WriteLine(a);

static void F(int a)
{
    a = 5;
}
```

**Ответы:**

* `a` внутри `F` — отдельная переменная в стеке.
* Внешняя `a` не изменится, на экран выведется `6`.

### Вопрос 3

```csharp
string a = "hello";
F(a);
Console.WriteLine(a);

static void F(string b)
{
    b = "world";
}
```

**Ответы:**

1. **Сколько объектов в динамической памяти выделится?**
   **Ответ:** Два. `"hello"` и `"world"` — строки в куче (immutable).
2. **Изменится ли объект, на который изначально ссылалась `a`?**
   **Ответ:** Нет. Строки неизменяемые.
3. **Изменится ли адрес переменной `a` на `"world"`?**
   **Ответ:** Нет. `b` — локальная копия ссылки. На экран выведется `"hello"`.

### Вопрос 4

```csharp
A b = new()
{
    value = 1,
};
F(b);
Console.WriteLine(b.value);

static void F(A a)
{
    a.value = 2;
}

struct A
{
    public int value;
}
```

**Ответы:**

1. **Изменится ли `value` внутри `b`?**
   **Ответ:** Нет. `struct` — значимый тип, передаётся по значению. Изменения внутри `F` не затронут `b`.

2. **Как изменится исход, если назвать переменную `b` как `a`?**
   **Ответ:** Ничего. Имя переменной не влияет на копирование значимого типа.

3. **Что будет, если сменить `struct` на `class`?**
   **Ответ:**

   * Теперь `A` — ссылочный тип.
   * `b` и `a` в `F` будут ссылками на один объект в куче.
   * Изменение `a.value = 2` изменит объект `b`, на экран выведется `2`.
   * Динамическая память выделится для самого объекта `A` в куче.
